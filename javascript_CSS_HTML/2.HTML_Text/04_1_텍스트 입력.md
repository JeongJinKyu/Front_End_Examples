### \<em> \<i>텍스트를 기울여주기

```<em></em>```태그는 emphasis의 줄임말,  ```<i></i>```태그는 italic의 줄임말<br>

```<em>``` 태그는 문장에서 흐름상 강조할 부분에 사용<br>
```<i>``` 태그는 마음속 생각, 용어, 관용구 등에 사용<br>

    일반 귤보다 2~3배 크고, 과육이 붉고 통통하다</p>
